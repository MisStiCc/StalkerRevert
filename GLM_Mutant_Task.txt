Спроектируй систему мутантов для игры "Сталкер наоборот" на Godot 4. 

Нужны:
- Базовый класс Mutant с общими параметрами и поведением
- Конкретные типы мутантов: Собаки, Снорки, Контролёры
- Параметры для каждого типа: здоровье, скорость, урон, броня, радиус обнаружения
- Поведение: патрулирование территории, атака на сталкеров, взаимодействие с Зоной
- Механика спавна мутантов за биомассу
- Сигналы для взаимодействия с ZoneController
- Особые способности каждого типа мутантов

Опиши подробно:
1. Структуру класса Mutant
2. Переменные и методы для каждого типа
3. Как мутанты взаимодействуют с ZoneController
4. Как работает патрулирование и атака
5. Систему стоимости спавна за биомассу
6. Сигналы для событий (смерть мутанта, атака, обнаружение сталкера)
