# Сталкер наоборот

Стратегия в реальном времени, где игрок управляет Зоной. Сталкеры приходят на территорию, чтобы украсть артефакты, а Зона защищается: создаёт аномалии, мутантов, управляет ресурсами.

## Концепция

Игра переворачивает традиционный подход к серии S.T.A.L.K.E.R.:
- **Игрок** — сама Зона
- **Цель** — не дать сталкерам унести артефакты
- **Механики** — создание аномалий, призыв мутантов, управление ресурсами, выбросы

## Разработка

Проект разрабатывается на Godot 4 с использованием GDScript.

### Структура проекта

- `assets/` - ресурсы (изображения, звуки, шрифты)
- `scenes/` - игровые сцены
- `scripts/` - скрипты игры
- `addons/` - Godot аддоны

### Технологии

- Godot 4
- GDScript
- GLM и Qwen для архитектурного проектирования и кодогенерации

## Критерии готовности

Игра считается готовой, когда:
- Зона создаётся, имеет ресурсы
- Сталкеры спавнятся, идут к артефактам
- Аномалии наносят урон
- Мутанты атакуют сталкеров
- Можно создавать аномалии за энергию
- Можно делать выброс
- Интерфейс показывает ресурсы
- Игра не падает с ошибками
